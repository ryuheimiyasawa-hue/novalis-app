# Philippine Community — 構想ドキュメント

## プロジェクト概要

**アプリ名**: Philippine Community（仮称）
**一言説明**: 日本在住フィリピン人の生活を安心・安全にサポートする総合プラットフォーム
**プラットフォーム形態**: LINE / SaaS / アプリ（未決定）

### 背景・目的

日本には約30万人のフィリピン人が在住しているが、言語の壁や制度の違いから、日常生活で多くのトラブルに直面している。労働問題、家庭内トラブル、行政手続きの困難さなど、相談先がわからず孤立するケースが多い。本プラットフォームは、AIを活用した一次対応と専門家（社労士・弁護士）のバックアップにより、フィリピン人コミュニティの生活基盤を包括的にサポートする。

### WHO / WHAT / HOW

| 項目 | 内容 |
|------|------|
| **WHO** | 日本在住のフィリピン人（就労者、配偶者、永住者、留学生等） |
| **WHAT** | 生活上の困りごとをワンストップで解決できるプラットフォーム |
| **HOW** | AIによる多言語一次対応 + 専門家エスカレーション + コミュニティ相互支援 |

## ユーザーが抱える具体的なトラブル事例

### 仕事のトラブル
- 日本語が話せないことに対するパワハラ
- 有給休暇を使わせてもらえない
- 社会保険料が給与から引かれているのに実際には未納
- 残業代の未払い
- 業務中の物損に対する不当な給与天引き
- 労災を使わせてもらえない
- 業務起因の疾病に対する補償なし
- 危険な労働環境での就労強制

### 夫婦・家庭のトラブル
- 不倫、家庭内暴力、経済的搾取
- 離婚拒否、勝手な離婚届提出
- ビザ剥奪の脅迫
- 夫婦間の言語の壁
- 親権問題
- 自宅からの追い出し

### 行政・市役所のトラブル
- 通訳サービスの地域格差
- 書類が読めない
- 低所得者向け制度の情報不足
- 高額療養費制度等の申請困難
- 学校関連書類への対応

### 警察関連のトラブル
- 詐欺被害（警察を名乗る者による）
- 犯罪経歴証明書の取得方法

### 緊急時のトラブル
- 家族の死亡時の対応手順
- いじめへの対処方法
- 高額医療費への対応
- 交通事故時の対処方法

## 技術構想

### AIの役割
- 多言語での一次対応（タガログ語、ビサヤ語、英語、日本語）
- FAQベースの自動回答
- トラブル内容の分類と適切な専門家へのルーティング
- 書類の翻訳・記入サポート

### エスカレーションフロー
```
ユーザーの相談
  → AIによる一次対応（情報提供・FAQ）
    → 解決 → 完了
    → 未解決 → 専門家（社労士/弁護士）へエスカレーション
      → 必要に応じて対面相談の手配
```
