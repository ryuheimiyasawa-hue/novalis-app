# Novalis デモ体験シナリオ — 協業企業様向け

**在日フィリピン人の生活支援サービス**

_バージョン: Beta (2026年5月)_

---

## ご挨拶

本日は **Novalis** のデモにご参加いただきありがとうございます。

Novalis は、在日フィリピン人の方々を対象とした **AI 相談サービス** です。在留資格、社会保険、子どもの教育、行政手続きなど、日本での生活に関する疑問にタガログ語で回答します。専門的判断が必要な相談は、提携する士業・専門家へエスカレーションする設計です。

本ドキュメントでは、**6つの体験シナリオ** を順にお試しいただき、実際のユーザー体験を確認していただきます。さらに、**協業可能性の検討に向けた4つの評価軸** での評価もお願いしております。

**所要時間**: 全シナリオで約 30〜40 分。

**アプリ URL**: `https://novalis-app.vercel.app`

> デモ用アカウントは事前にご案内します。Facebook ログインを使用しますので、貴社用のアカウントでログインしていただくか、Novalis 側でご用意した検証アカウントをご利用ください。

## シナリオ A — 在留資格更新について質問する

### 想定ユーザー

在留期限が 3 ヶ月以内に迫っている、就労ビザ保持のフィリピン人労働者。

### 試す手順

1. `https://novalis-app.vercel.app` を開き、Facebook でログインします。
2. ダッシュボードから **「相談を始める」** をクリックします。
3. チャット入力欄に次の質問を入力します: _「就労ビザの更新に必要な書類を教えてください」_
4. **「送信」** を押し、AI の回答を待ちます。
5. 回答の下に表示される **引用記事 (citations)** のリンクを 1 つクリックします。

### 期待される体験

- 10〜30 秒以内にタガログ語で具体的な書類リストが返答されます。
- 回答末尾に **引用元の社内ナレッジ記事** へのリンクが表示され、根拠を確認できます。
- 利用者は **Google 検索 (日本語) を回避しつつ、母国語で短時間で要件を把握** できます。

## シナリオ B — 過去の相談履歴を確認する

### 想定ユーザー

数日前に Novalis で相談したことのある利用者が、その続きから会話を再開したい場面。

### 試す手順

1. 事前にシナリオ A を完了している前提です (未完了の場合は先に A を実施)。
2. チャット画面に移動します。
3. **左サイドバー** に過去の相談履歴が一覧表示されていることを確認します。
4. 一覧から先ほどの「在留資格更新」の相談をクリックし、過去のメッセージ全文を確認します。
5. 同じスレッドで **追加の質問** (例:「申請場所は最寄りの入管でよいですか?」) を投げます。

### 期待される体験

- 過去の全メッセージが同じ順序で表示されます。
- AI は過去文脈を踏まえた回答を返し、ユーザーが背景を再説明する必要がありません。
- スマートフォン環境ではサイドバーは折りたたまれ、ハンバーガーアイコンで展開できます。

## シナリオ C — 個別事案のエスカレーション

### 想定ユーザー

直近に転職したフィリピン人労働者で、就労ビザに関する具体的な手続き判断を求めている。

### 試す手順

1. チャット画面で **新規の相談** を開始します。
2. 次の質問を入力します: _「先週、現在の会社を退職して別の会社に転職しました。就労ビザについて何をすべきですか?」_
3. **「送信」** を押し、AI の応答を確認します。
4. **エスカレーション画面** が表示され、専門家リストと「Novalis サポートに問い合わせる」ボタンが見えることを確認します。
5. **「Novalis サポートに問い合わせる」** をクリックし、遷移先 (Google Form) を確認します。

### 期待される体験

- 個別性の高い法的判断を伴う質問では、AI は **直接回答せず** 専門家への接続を提示します。
- これは **弁護士法・行政書士法等への配慮** であり、ユーザー保護の観点で重要な仕様です。
- 単なる AI 一発回答では実現できない、Novalis の差別化ポイントです。

## シナリオ D — ナレッジ記事を閲覧する

### 想定ユーザー

時間のあるときに、社会保険や国民健康保険などの基礎情報を体系的に学びたい利用者。

### 試す手順

1. ダッシュボードから **「記事一覧を見る」** または **「役立つ情報」** をクリックします。
2. 記事一覧でカテゴリ (Visa, Healthcare, Childcare 等) を確認します。
3. 社会保険または国民健康保険に関する記事を 1 件選択し、本文を閲覧します。
4. **埋め込み動画** がある場合は再生し、フルスクリーン表示も確認します。
5. スマートフォンでも開いてみて、レスポンシブ表示を確認します。

### 期待される体験

- タガログ語で、読みやすい構造化された記事が表示されます。
- 動画埋め込み記事では YouTube / Vimeo プレイヤーがネイティブに再生されます。
- スマートフォンでも文字の崩れや動画の表示崩れがありません。

## シナリオ E — 多言語切替

### 想定ユーザー

英語の方が技術用語を理解しやすい利用者、または日本人の同僚に画面を見せたい場面。

### 試す手順

1. 任意の画面の右上にある **言語切替** をクリックします。
2. **English** を選択し、UI 全体が英語に切り替わることを確認します。
3. チャットで新規質問を投げ、AI の回答が英語で返ることを確認します。
4. **日本語** を選択し、同じ確認を行います。
5. **Tagalog** に戻します。

### 期待される体験

- UI の全テキストが即座に切り替わります (リロード不要)。
- AI 回答も選択言語に追従します。
- レイアウト崩れや翻訳漏れがありません。

## シナリオ F — トラブル時の問い合わせ動線

### 想定ユーザー

バグや使い方の質問など、AI ではなく Novalis 運営チームへ直接連絡したい場面。

### 試す手順

1. シナリオ C のエスカレーション画面、もしくは直接 `https://novalis-app.vercel.app/ja/contact` を開きます。
2. **「Novalis サポートに問い合わせる」** をクリックします。
3. 新しいタブで **Google Form** が開くことを確認します。
4. 名前、メール、**カテゴリ (Bug / Question / Feedback)**、メッセージを入力します。
5. フォームを送信し、確認ページが表示されることを確認します。

### 期待される体験

- 問い合わせ動線がどの画面からでも 2 クリック以内で到達できます。
- フォーム本体は Google Form で、Novalis 運営チームのメールに自動送信されます。
- 将来的には first-party のお問い合わせ管理に置き換えますが、Beta 期間は Google Form で運用します。
